# Sentiment Analyzer

A sleek, beginner-friendly web application that analyzes the emotional tone of text.

## Features
- **Modern Design:** Dark mode with glassmorphic cards and floating glow blobs.
- **Instant Analysis:** Classifies input as Positive (😊), Negative (😢), or Neutral (😐).
- **Sentiment Score:** Shows a numeric score and shifts a scale indicator.
- **Keyword Breakdown:** Shows lists of matched positive and negative words.
- **Text Highlighting:** Colors positive words in green and negative words in red inside the text.

## Technologies Used
- HTML5
- CSS3 (Vanilla transitions and variables)
- JavaScript (Vanilla DOM & RegExp matching)
